Développeurs

Cyan Worlds, Inc. a accepté d'ouvrir les codes sources pour Myst Online: Uru Live. Seront inclus les codes sources client, serveurs et outils. Grâce aux codes sources client et serveurs, les fans devraient être en mesure d'installer et de faire fonctionner Myst Online: Uru Live et permettre ainsi le retour en ligne de la communauté d'Uru. Avec le code source outils, qui inclut le plugin Plasma30 3ds Max, les fans devraient pouvoir créer de nouveaux âges ainsi que de nouveaux outils pour d'autres programmes de modélisation en 3D.

Les modèles 3D et les textures qui ont été utilisés pour créer Myst Online: Uru Live ne seront pas inclus dans la licence code source ouverte. Nous allons probablement mettre à la disposition des fans certains de ces modèles et textures, mais sous une licence différente de celle du code source ouverte. Toutefois, nous fournirons un serveur Myst Online: Uru Live incluant une compilation des données du jeu afin que les serveurs tenus par les fans puissent s'y connecter pour ensuite retransmettre ces données à leurs usagers.

Pour l'instant, nous préparons les sources pour le code source ouverte pendant notre temps libre. Même si le "nous" inclut aussi quelques ex-programmeurs de Cyan, cela reste un travail long et fastidieux. Si l'une des sources s'avérait prête plus rapidement, peut-être la rendrons-nous publique avant les autres. Vous pourrez alors y accéder par un lien URL qui sera ajouté ici.
