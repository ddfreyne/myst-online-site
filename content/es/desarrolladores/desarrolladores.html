Desarrolladores

Cyan Worlds, Inc. ha aceptado poner el código fuente de Myst Online: Uru Live en modalidad de código abierto. Las fuentes que serán incluidas son el código para el cliente, todos los servidores y las herramientas. Con el código fuente del cliente y los servidores, los fans serán capaces de instalar y ejecutar Myst Online: Uru Live y llevar la comunidad de Uru de vuelta en línea. Con el código de las herramientas que incluye el plugin Plasma20 3ds Max los fans serán capaces de crear nuevas Eras así como crear nuevas herramientas para otros programas de modelado 3D.

Los modelos 3D y texturas que fueron usadas para crear Myst Online: Uru Live no serán parte de la licencia de código abierto. Probablemente publicaremos algunos de estos modelos y texturas, pero bajo una licencia distinta que la de código abierto. De todos modos, crearemos un servidor de datos para Myst Online: Uru Live con los datos compilados del juego para que los servidores mantenidos por fans puedan conectarse y servir esos archivos a sus usuarios.

Actualmente, estamos preparando las fuentes para el código abierto en nuestro tiempo libre. Con "nosotros" nos referimos también a unos cuantos ex-programadores de Cyan, pero es una tediosa tarea que lleva tiempo. Si un paquete de fuentes va más rápido, podemos publicarlo antes que los demás. Cuando publiquemos algo, será en un repositorio de código abierto. El enlace a ese repositorio de código abierto se añadirá aquí.
